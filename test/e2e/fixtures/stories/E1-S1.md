# E1-S1: Add Todo

## Description
User can add a new todo item via the CLI.

## User Story
As a user, I want to add a todo item so that I can track tasks.

## Acceptance Criteria
1. `node todo.js add "buy milk"` creates a todo with auto-incremented ID
2. New todo has completed: false and createdAt timestamp
3. Todo is persisted to todos.json
4. Command prints the new todo ID to stdout
5. Missing text argument returns exit code 1
