# E1-S4: Delete Todo

## Description
User can delete a todo item.

## User Story
As a user, I want to delete a todo so that I can remove tasks I no longer need.

## Acceptance Criteria
1. `node todo.js delete 1` removes the todo with id 1
2. Deleted todo is removed from todos.json
3. Non-existent ID returns exit code 1 with error message
