# E1-S3: Complete Todo

## Description
User can mark a todo as completed.

## User Story
As a user, I want to complete a todo so that I can track progress.

## Acceptance Criteria
1. `node todo.js complete 1` sets completed: true for todo with id 1
2. Completed todos show a checkmark in list output
3. Invalid ID returns exit code 1 with error message
