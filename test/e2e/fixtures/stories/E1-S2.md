# E1-S2: List Todos

## Description
User can list all todo items in a formatted table.

## User Story
As a user, I want to list my todos so that I can see what needs to be done.

## Acceptance Criteria
1. `node todo.js list` displays all todos in a table
2. Table shows id, status checkbox, text, and age
3. Output fits in 80-column terminal
4. Empty list shows a message (no error)
